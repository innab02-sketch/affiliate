# סוכן אוטומטי לסריקת הצעות ClickBank 🤖

סוכן זה סורק מדי יום את ClickBank Marketplace, מחלץ את ההצעות המובילות בקטגוריות נבחרות, מנקד אותן בעזרת Claude AI (לפי מודל משקלים מותאם אישית), ושומר את ההצעות המצטיינות (ציון 50+) ב-Google Sheets. בנוסף, הוא יכול לשלוח סיכום יומי למייל.

---

## תכונות מרכזיות
- **עקיפת הגנות (Anti-Bot):** שימוש ב-Playwright (דפדפן אמיתי) כדי למנוע חסימות מצד ClickBank.
- **ניקוד חכם:** מודל מבוסס משקלים (EPC, עמלה, מנוי חוזר, Gravity, יציבות).
- **ניתוח איכותני AI:** שימוש ב-Claude 3.5 כדי לספק משוב של 2-3 משפטים בעברית על כל הצעה.
- **אוטומציה מלאה:** ייצוא אוטומטי ל-Google Sheets ושליחת התראות במייל.

---

## דרישות מוקדמות
- שרת Linux (מומלץ Ubuntu 22.04 / 24.04).
- Python 3.10 ומעלה.
- חשבון [Anthropic](https://console.anthropic.com/) למפתח API של Claude.
- חשבון Google Cloud ליצירת Service Account (עבור Google Sheets).

---

## שלב 1: התקנה על השרת

1. **העתקת הקבצים:** העבירי את כל תיקיית הפרויקט לשרת שלך (למשל תחת `/home/ubuntu/clickbank_scanner`).
2. **הרצת סקריפט ההתקנה:**
   נווטו לתיקייה והריצו את סקריפט ההתקנה שייצור סביבה וירטואלית ויתקין את כל התלויות (כולל דפדפני Playwright):
   ```bash
   cd /home/ubuntu/clickbank_scanner
   chmod +x setup.sh
   ./setup.sh
   ```

---

## שלב 2: הגדרת Google Sheets API

כדי שהסקריפט יוכל לכתוב נתונים לגיליון שלך, עליו להזדהות מול Google.

1. כנסי ל-[Google Cloud Console](https://console.cloud.google.com/).
2. צרי פרויקט חדש (או השתמשי בקיים).
3. חפשי בשורת החיפוש העליונה **"Google Sheets API"** ולחצי על **Enable** (הפעלה).
4. חפשי גם **"Google Drive API"** ולחצי על **Enable**.
5. בתפריט הצד, נווטי ל-**APIs & Services** -> **Credentials**.
6. לחצי על **Create Credentials** ובחרי ב-**Service Account**.
7. תני לו שם (למשל `clickbank-bot`) וסיימי את היצירה.
8. ברשימת ה-Service Accounts, לחצי על המייל של החשבון שנוצר.
9. נווטי ללשונית **Keys**, לחצי על **Add Key** -> **Create new key**, ובחרי בפורמט **JSON**.
10. הקובץ יירד למחשב שלך. **שני את שמו ל-`service_account.json`** והעלי אותו לתיקיית הפרויקט בשרת (לצד `main.py`).
11. **חשוב מאוד:** פתחי את ה-Google Sheet שלך בדפדפן, לחצי על **Share** (שיתוף), והוסיפי את כתובת המייל של ה-Service Account (הכתובת הארוכה שמסתיימת ב-`@...iam.gserviceaccount.com`) כ-**Editor** (עורך).

---

## שלב 3: הגדרת משתני סביבה (קונפיגורציה)

אמנם ניתן לערוך ישירות את הקובץ `config.py`, אך מטעמי אבטחה (במיוחד עבור מפתחות API), מומלץ להגדיר אותם כמשתני סביבה.

צרי קובץ בשם `.env` בתיקיית הפרויקט (אופציונלי, נדרש לטעון אותו ב-cron), או הגדירי ישירות ב-`config.py`:

**הגדרות חובה ב-`config.py` (או כמשתני סביבה):**
- `ANTHROPIC_API_KEY`: המפתח של Claude.
- `GOOGLE_SHEET_ID`: ה-ID של הגיליון (החלק הארוך ב-URL של הגיליון שלך).

*הערה: ראי את קובץ `config.py` להסבר מפורט על כל פרמטר ופרמטר.*

---

## שלב 4: בדיקה ידנית

לפני שמגדירים אוטומציה, כדאי לוודא שהכל עובד:
```bash
cd /home/ubuntu/clickbank_scanner
source venv/bin/activate
python main.py
```
בדקי את הקונסול ואת תיקיית ה-`logs/` כדי לוודא שאין שגיאות ושהנתונים אכן נכתבו ל-Google Sheet.

---

## שלב 5: הגדרת אוטומציה יומית (Cron Job)

כדי שהסקריפט ירוץ אוטומטית כל יום (למשל ב-08:00 בבוקר), נוסיף משימה ל-cron.

1. פתחי את עורך ה-cron:
   ```bash
   crontab -e
   ```
2. הוסיפי את השורה הבאה בסוף הקובץ (הקפידי לעדכן את הנתיבים אם שמת את הפרויקט במיקום אחר):
   ```bash
   0 8 * * * cd /home/ubuntu/clickbank_scanner && /home/ubuntu/clickbank_scanner/venv/bin/python /home/ubuntu/clickbank_scanner/main.py >> /home/ubuntu/clickbank_scanner/logs/cron.log 2>&1
   ```
   *(שורה זו תריץ את הסקריפט כל יום ב-08:00 בבוקר, ותשמור פלטי שגיאות מערכת לקובץ `cron.log`)*

## פתרון תקלות נפוצות

- **"כשל בכתיבה ל-Google Sheet"**: ודאי ששיתפת את הגיליון (Share) עם כתובת המייל של ה-Service Account, ושה-`GOOGLE_SHEET_ID` מוגדר נכון.
- **"לא נשלפו מוצרים כלל"**: ייתכן ש-ClickBank שינו את מבנה ה-HTML של האתר. בדקי את צילומי המסך שנוצרים בתיקיית `logs/` (למשל `debug_home-garden.png`) כדי לראות מה הדפדפן ראה.
- **שגיאות Claude API**: ודאי שיש לך קרדיטים/יתרה בחשבון ה-Anthropic שלך ושהמפתח מוגדר נכון. (שים לב: גרסת `anthropic>=0.49.0` נדרשת כדי למנוע קונפליקטים עם ספריות אחרות כמו Twilio).
